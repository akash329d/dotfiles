# dotfiles

Terminal config and setup scripts using [Sheldon](https://github.com/rossmacarthur/sheldon) for plugin management.

### Installation

```console
curl -sL https://raw.githubusercontent.com/akash329d/dotfiles/main/setup.zsh | zsh
```

### What's Installed

**Plugins** (via Sheldon):
- powerlevel10k, fzf-tab, zsh-autosuggestions, zsh-syntax-highlighting, zsh-alias-finder

**CLI Tools** (via Homebrew):
- fzf, bat, eza, gitui, jless, zoxide, atuin, ripgrep

### Update

```console
dfu
```
